About Website

What are the uses of this Periodic Table website?

  1. find elements data (atomic number, appearance, name, symbol, category, atomic mass, melting point, boiling point, phase , density, molar heat, etc.)
  2. create a link for a certain element
  3. see phases of elements in different temperatures
  4. find elements by name or atomic number, easily
  5. find charts of electronegativities and ionization energies

How was this made?

I used jQuery, Font Awesome, PeriodicTableJSON(database of elements) and Charts.js as plugins.

The web pages are made with HTML, CSS and JavaScript. I used a JSON file to get elements data. -> Periodic-Table-JSON github link.

To create the charts page we used Charts.js. github link.
