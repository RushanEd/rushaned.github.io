About

      Online periodic table was made to help chemistry students, teachers and anyone who like it. The idea to this website is based on periodic table created by Dmittri Mendelyeev. It's a chemical elements classification system based on element's atomic number. Lastly updated on 22nd of December, 2018 with information of 126 elements. We used Wikipedia, Periodic Table(android app) and Periodic Table(windows software) as sources of information.
